# ToDoアプリ（期限通知メール・視覚強調機能付き）

## 概要

本アプリは、Python（Tkinter）を用いて開発したデスクトップ型ToDo管理アプリです。
タスクの期限管理に加え、**期限1日前のメール通知機能**および**期限間近タスクの赤文字表示**を実装し、タスクの見落とし防止を目的としています。

---

## 主な機能

* ユーザー登録・ログイン機能
* タスクの追加 / 完了 / 削除
* 期限日時の設定（カレンダー + 時刻入力）
* **期限1日前のメール通知（Gmail SMTP）**
* **期限24時間以内の未完了タスクを赤文字表示**
* ユーザーごとのタスク管理（JSONによる保存）

---

## 使用技術

* Python 3
* Tkinter（GUI）
* tkcalendar（カレンダーUI）
* JSON（データ保存）
* smtplib / email（メール送信）
* Gmail SMTP

---

## システムの流れ

```text
ユーザー操作
   ↓
Tkinter GUI
   ↓
タスク登録（JSON保存）
   ↓
1分ごとの期限チェック
   ↓
期限24時間以内
   ↓
メール送信 + 赤文字表示
```

---

## 工夫した点

###  期限通知の重複防止

タスクに `notified` フラグを持たせることで、
**同じタスクに対して複数回通知が送られない設計**にしました。

---


###  視覚的な優先度表示

期限24時間以内の未完了タスクを赤色で表示することで、
**ユーザーが直感的に重要タスクを認識できるUI設計**としました。

---

### ユーザー単位のデータ管理

```json
{
  "user@example.com": [
    {
      "text": "タスク内容",
      "deadline": "2026-04-30 10:00"
    }
  ]
}
```

の形式で管理し、**ユーザーごとにタスクを分離**しています。

## セットアップ方法

### 1. ライブラリのインストール

```bash
pip install tkcalendar python-dotenv
```

---

### 2. 環境変数の設定

`.env` ファイルを作成

```env
SENDER_EMAIL=your-email@gmail.com
SENDER_PASSWORD=your-app-password
```

※ Gmailは「アプリパスワード」を使用

---

### 3. 起動

```bash
python main.py
```

---

## 使い方

1. 新規登録（メールアドレス・パスワード）
2. ログイン
3. タスク名と期限日時を入力
4. タスクを追加
5. 期限1日前にメール通知
6. 期限が近いタスクは赤文字で表示

## 面接での説明ポイント

* なぜJSONを使用したか
  → 軽量で扱いやすく、小規模アプリとして適しているため

* なぜ1分間隔でチェックしたか
  → リアルタイム性と処理負荷のバランスを考慮

* メール送信の仕組み
  → SMTP + TLSで暗号化通信し、Gmailサーバー経由で送信
